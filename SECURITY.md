# Security Policy

This repository is a public-safe educational and portfolio demo.

It must never contain:

- Real tick data
- Broker credentials
- Database URLs
- API keys or tokens
- Live account information
- Private AlphaQuant V12 schema
- Private strategy thresholds
- Production execution logic
- Model files
- Real trading logs
- Database dumps

## Public Demo Rule

All examples must remain synthetic, simplified, and non-operational.

This repository demonstrates PostgreSQL schema design, market-data ingestion structure, SQL validation, and testing discipline. It is not financial advice and must not be used as a live trading system.

## Reporting a Security Issue

If sensitive information is accidentally found in this repository, contact the maintainer immediately.
